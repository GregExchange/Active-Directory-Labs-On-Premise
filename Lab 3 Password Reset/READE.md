## LAB 3 Password Reset Ticket

## Objective
Simulated a real-world Helpdesk ticket where a user forgot their password.

## Steps
1. Open Active Directory Users and Computers console.
2. Find User-Reset Password window
3. User Password reset confirmation
4. Successful login on client machine

Used Active Directory Users and Computers to locate the user account,
reset the password, unlock the account, and verify successful login
from a Windows client machine.
