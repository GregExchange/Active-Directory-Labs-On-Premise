## LAB 4 Password Reset Ticket

## Objective
Simulated a real-word Helpdesk Ticket where a user forgot their password.

## Steps
1. Open Active Directory Users and Computers console.
2. Find User-Reset Password window.
3. User Password reset confirmation
4. Successful login on client Machine

Used Active Directory Users and Computers to locate the user account, reset the password, unlock the account, and verify from a Windows client machine.
