## Objective
Create a shared folder and control access using Security Groups

## Scenario
Department: HR
Request: HR team needs a shared folder.
Only HR users should have access.

## Steps
1. Create Security Group.
2. Add Users to Group.
3. Create the Shared Folder.
4. Configure Share Permissions.
5. Configure NTFS Permissions.
6. Access Shared Folder from Client.
7. Test Access (Authorized User).
8. Step 8 — Test Access (Unauthorized User).


Configured a secure shared network drive using Active Directory
security groups. Created an HR group, assigned users, configured
share and NTFS permissions, and validated access control from a
Windows 11 client machine.
